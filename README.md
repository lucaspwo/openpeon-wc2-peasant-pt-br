# WC2 Peasant (PT-BR)

Pack de sons do camponês humano do WarCraft II dublado em português brasileiro, no formato [OpenPeon/CESP v1.0](https://openpeon.com/spec).

## Instalação

```bash
peonping install wc2_peasant_pt-br
```

## Categorias

| Categoria | Sons | Descrição |
|---|---|---|
| `session.start` | 5 | Saudação ao iniciar sessão |
| `task.acknowledge` | 4 | Confirmação de tarefa |
| `task.complete` | 1 | Tarefa concluída |
| `task.error` | 1 | Reação a erro |
| `input.required` | 4 | Esperando input do usuário |
| `resource.limit` | 1 | Recurso no limite |
| `user.spam` | 7 | Cliques excessivos |
